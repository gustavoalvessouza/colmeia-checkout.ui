# comeia-checkout.ui

## how to run the project

-   clone this repository
-   run `npm install`
-   run `npm run dev`

## to improve dev experience

-   install .vscode/ recommended extesions.

## used techs

-   Next.js
-   Tailwind
-   Biome (Linter and Code formatter)
-   Indexed DB (Simulate Database)
-   react-hook-form (Uncontrolled forms)
-   Sooner (toast alerts)
